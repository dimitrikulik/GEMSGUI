GEM-Selektor version 2-PSI

A Gibbs Energy Minimization program package for interactive thermodynamic modeling of heterogeneous aquatic (geo)chemical systems, especially those involving metastability and dispersity of mineral phases, solid solution - aqueous solution equilbria, and adsorption/ion exchange. Includes a built-in (default) thermodynamic database in both thermochemical and reaction formats. Processes of irreversible chemical mass transfer can be simulated from the principles of local and partial equilibrium.

GEM-Selektor version 2-PSI (GEMS-PSI) code is written in C/C++, it uses the Qt3.x GUI Toolkit (Trolltech AG, Norway, http://www.trolltech.com) which makes GEMS a cross-platform software for Linux/Unix and Windows-32 systems. The GEMS executable code is obtained by linking together two parts, or libraries:

GEM-Selektor (gemsel.lib, gemsel.o) contains all the numerical modules written in C/C++ by D.Kulik, K.Chudnenko, I.Karpov, S.Dmytriyeva, A.Rysin and others using earlier MS-DOS program Selektor-A. This part can be compiled without Qt headers and documentation (i.e. developed without Qt commercial license) using an ANSI-compliant C++ compiler.

GEM-Vizor (gemviz.lib, gemviz.o) contains all GUI and DBMS modules, widgets and dialogs, written  by S.Dmytriyeva and A.Rysin using an older MS DOS version by D.Kulik and S.Dmytriyeva. This part (developed under the Qt-Duo enterprise licenses #1435515and #1435740) uses many Qt3.x classes and requires Qt headers for compilation, as well as qtmt.dll or libqt-mt for run time.
