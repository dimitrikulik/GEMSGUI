Automatic Setup of Aqueous and Gas/Fluid Phases

This dialog facilitates setup of definitions of the aqueous electrolyte phase and the gas/fluid mixture phase that will be automatically assembled when opening (or creating) a GEMS modelling project. For the underlying models and equations, see the theorgem.html manual and pdf manual files linked to it.

The first tab "Select Aqueous Electrolyte Model" lets you define a built-in function for calculating activity coefficients of individual aqueous species. These can be selected by clicking on radio-buttons on the left side, or pressing  Alt-D, Alt-H, Alt-3, ... key combinations. Two last choices should be used when none of the built-in functions is applicable, or no aqueous phase should be included into the chemical system definition. In the "Parameters" frame, you can change some default parameters used in some of the built-in aqueous activity functions. 

Clicking on the "Check" button displays a record key of the Phase record to be automatically generated;  the chosen model and its parameters will be identified in the PhName and PhNote fields. You can check them later when you open the "Phase" page of the "System" window from the "Single Thermodynamic System" dialog, mark the phase record key and press F7. Note that, when the same modelling project is opened next time, this Phase definition record will be automatically deleted and generated again without displaying this setup dialog.

The second tab "Select Gas/Fluid Mixture Model" defines the automatically generated Phase record for gas or fluid mixture. It functions in a way similar to that for the aqueous phase. Note that no Parameters for the gas/fluid mixture (on the right side) are used in this version of GEM Selektor.

The setup of automatically generated Phase records for aqueous and gas phases can be changed any time upon opening this modelling project. To do this, unclick the "Retain setup of aqueous (and gas) phases" in the "Modelling Projects" dialog before selecting in the list an exitsting project to be opened.
